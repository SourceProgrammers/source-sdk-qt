# Source SDK Qt
Source SDK Qt is a [Qt](https://www.qt.io/) implementation for the Source SDK, currently compatable with Source SDK 2013.
<br>
Required stuff:
- SDK: Source SDK 2013 MultiPlayer
- OS: Windows 7+
